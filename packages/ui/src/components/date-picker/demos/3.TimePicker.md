# en-US

Configure the time picker via `dShowTime`.

# zh-Hant

通过 `dShowTime` 配置时间选择器。

```tsx
import { DDatePicker } from '@react-devui/ui';

export default function Demo() {
  return (
    <>
      <DDatePicker dShowTime />
      <DDatePicker dShowTime={{ d12Hour: true }} />
    </>
  );
}
```
