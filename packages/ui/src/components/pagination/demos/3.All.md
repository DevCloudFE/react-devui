# en-US

Configure freely through `dCompose`.

# zh-CN

通过 `dCompose` 自由组合配置。

```tsx
import { DPagination } from '@react-devui/ui';

export default function Demo() {
  return <DPagination dTotal={200} dCompose={['total', 'pages', 'size', 'jump']}></DPagination>;
}
```
