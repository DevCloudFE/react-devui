## API

### DComposeProps

继承 `React.HTMLAttributes<HTMLDivElement>`。

<!-- prettier-ignore-start -->
| 参数 | 说明 | 类型 | 默认值 | 
| --- | --- | --- | --- | 
| dSize | 设置子项尺寸 | 'smaller' \| 'larger' | - |
| dDisabled | 是否禁用子项 | boolean | false |
<!-- prettier-ignore-end -->

### DComposeItemProps

继承 `React.HTMLAttributes<HTMLDivElement>`。

<!-- prettier-ignore-start -->
| 参数 | 说明 | 类型 | 默认值 | 
| --- | --- | --- | --- | 
| dGray | 常用的偏灰背景 | boolean | false |
<!-- prettier-ignore-end -->
