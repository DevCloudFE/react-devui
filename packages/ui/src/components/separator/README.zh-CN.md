## API

### DSeparatorProps

```tsx
interface DSeparatorProps extends React.HTMLAttributes<HTMLElement> {
  dTextAlign?: 'left' | 'right' | 'center';
  dVertical?: boolean;
}
```

<!-- prettier-ignore-start -->
| 属性 | 说明 | 默认值 | 版本 | 
| --- | --- | --- | --- | 
| dTextAlign | 设置文字的位置 | `'left'` |  |
| dVertical | 设置垂直分隔 | `false` |  |
<!-- prettier-ignore-end -->
